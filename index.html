Pathfinding Demo

A simple example of pathfinding in a side on world taken from the project is shown here. The algorithm divides the problem into macro and micro-level planning stages in order to discretize the effectively continuous space through which characters move. For the macro-level planning an A* search algorithm is implemented appealing to basic mechanics in order to determine the possible new positions from any given tile. Micro-level planning is taken care of by the Enemy object which attempts to follow the path found by the maco-level planner. For this reason it is occasionally possible for Enemy to fall off of its intended path, this becomes increasingly more likely in the context of Enemy-Player interactions and so more focus will be placed on recovery rather than strictly adhering to the calculated path.
